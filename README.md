# 🏀 BasketPredictAI

Plateforme de prédiction de matchs basketball
NBA · EuroLeague · NCAA · BSL · LNB

## Stack
- **Backend** : FastAPI + Python
- **Frontend** : Streamlit
- **IA** : Google Gemini 1.5 Flash
- **Recherche** : Serper.dev + Jina AI Reader
- **Calculs** : NumPy · SciPy · Pandas
- **Déploiement** : Render + Streamlit Cloud

## Fonctionnalités
- Prédiction par Elo Dynamique
- Four Factors (Dean Oliver)
- 20 000 simulations Monte Carlo
- Détection Value Bet
- Rapport blessures en temps réel
- Analyse narrative Gemini AI

## Ligues supportées
NBA · EuroLeague · NCAA · BSL · LNB
